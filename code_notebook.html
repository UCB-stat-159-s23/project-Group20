Stat 159 Group 20 Project#

Housing Prices Prediction#

Introduction#

This project explores factors that impact the housing price, such as demographic, socioeconomic, and environmental factors. The dataset is obtained from a Kaggle tutorial competition. The study starts with a introduction with a brief overview of the project’s objectives and the data sources used in the analysis. Then, we did data import and data cleaning. We created a 70/30 training/test dataset. For exploratory data analysis (EDA), we conducted descriptive statistics and visualizations, such as bar charts, histograms, and scatter plots. With the plots from EDA, we did feature selection to select the most relevant features for modeling using correlation analysis, feature importance techniques, and domain knowledge.

We also conduct a simple ordinary least squares (OLS) regression to predict the outcome of interest using a subset of features. Then, we compared model performance using evaluation metrics such as R-squared, mean squared error, and mean absolute error. We provided the necessary packages and tests for this project, as well as the datasets and plots.

Feature Correlation Heatmap#

We use a heatmap to see which numeric variables have a high correlation with SalePrice.

# Create a copy of the numerical variables DataFrame
all_numVar = data_cleaned_numerical.copy()

# Compute correlations of all numerical variables
cor_numVar = all_numVar.corr() 

# Sort on decreasing correlations with 'SalePrice'
cor_sorted = cor_numVar['SalePrice'].sort_values(ascending=False)

# Select only high correlations (a threshold of 0.6)
CorHigh = cor_sorted[abs(cor_sorted) > 0.6].index

# Subset the correlation matrix for high correlations only
cor_numVar = cor_numVar.loc[CorHigh, CorHigh]

# Plot the correlation matrix using a heatmap
heatmap = sns.heatmap(cor_numVar, annot=True, cmap='coolwarm', center=0)
plt.xticks(rotation=45) 
plt.yticks(rotation=0)  

plt.savefig('figures/heatmap.jpg')  # Save the figure as an image
plt.show()

This heatmap shows the six variables that have a high correlation with the response variable SalePrice, suggesting their potential importance in predicting house sale prices.

From the plot, we can observe that some variables are highly correlated with each other. For instance, the correlation between GarageArea and GarageCars is extremely high with a correlation coefficient of 0.88. This high correlation suggests the presence of multicollinearity between these two variables.

Ordinary Least Squares (OLS) Regression#

training_data, testing_data = train_test_split(data_cleaned_numerical, test_size=1/3, random_state=0)

ols = smf.ols(formula = 'SalePrice ~ OverallQual + GrLivArea + GarageCars + \
                        GarageArea + TotalBsmtSF + Q("1stFlrSF")', 
              data = training_data).fit()

The OLS OSR2 value of approximately 0.694 indicates that the OLS regression model explains approximately 69.4% of the variation in the sale prices of homes in the testing set. In other words, the selected independent variables (OverallQual, GrLivArea, GarageCars, GarageArea, TotalBsmtSF, 1stFlrSF) collectively contribute to explaining about 69.4% of the variability in the sale prices.

The OLS MAE value of approximately 25351.735 represents the average absolute difference between the predicted sale prices and the actual sale prices in the testing set. This indicates that, on average, the OLS regression model’s predictions deviate from the actual sale prices by approximately $25,351.735.
